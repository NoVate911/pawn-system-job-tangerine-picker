<h2 align="center">Система работы сборщика мандаринов для SA:MP и CR:MP.</h2>
<p align="center"><sup>Последняя дата обновления репозитория: 07.10.2022</sup></p>
<h2>Описание</h2>
<p>Данная система работы представляет собой сборку мандаринов. После того как игрок подходит к дереву и собирает мандарины, то с каким-то шансом ему выпадает мандарин. Также есть капча, которая проверять игрока на бота. Система работает как на платформе SA:MP, а также в CR:MP и не требует никакой адаптации.</p>
<h2>Начало и установка</h2>
<h3>Софт для старта</h3>
<p>В самом начале нужно установить программы и настроить их.</p>
<p><sup>Под символом * обязательный софт.</sup></p>
<ul>
<li><a href="https://code.visualstudio.com/sha/download?build=stable&os=win32-x64-user">Install Visual Studio Code [Edit Code]</a></li>
<li><a href="https://github.com/git-for-windows/git/releases/download/v2.37.3.windows.1/Git-2.37.3-64-bit.exe">Install Git</a></li>
</ul>
<h3>После установки софта</h3>
<p>Открываем консоль (ПКМ) и вставляем туда следующую команду.</p>
<p><sup>Без Git работать не будет!</sup></p>

```
git clone https://github.com/NoVate911/pawn-system-job-tangerine-picker.git
```
- После копирования репозитория в папку, откройте её и возьмите оттуда файл **.pwn**, который лежит в папке **filterscripts** и переместите его в свою сборку.
- После перемещения, откройте файл **server.cfg** и в строку **filterscripts** пропишите имя файла **.pwn** и сохраните.
- Зайдите в папку **filterscripts** и запустите перемещённый файл, а после чего скомпилируйте его.
